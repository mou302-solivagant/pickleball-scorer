# 匹克球計分器 (Pickleball Scorer)

一個專為匹克球(Pickleball)設計的網頁計分器,可安裝成手機 App(PWA),讓場邊的朋友輕鬆擔任裁判計分。

🔗 **線上使用:** https://pickleball-scorer-livid.vercel.app

---

## 功能特色

- **一鍵計分**:裁判只需點選「贏得該球的隊伍」,系統自動判斷加分或轉換發球權,不需要自己判斷複雜規則
- **雙彈跳規則邏輯**:內建正確的發球權轉換機制
  - 每局第一次發球(整場比賽開局)固定從 `0:0:2` 開始,發球方只有一次機會,接發球方一得分即整隊出局換邊
  - 之後每次換邊發球,新發球隊會有兩次機會(第 1 發球員失分 → 換第 2 發球員 → 再失分才整隊出局)
- **標準報分格式**:畫面即時顯示 `發球方分數 – 接發球方分數 – 發球員編號`,方便裁判口頭唱名
- **重發球(Let)**:遇到需要重來的球,不影響分數與發球權
- **復原**:誤按可以退回上一步
- **彈性局制**:可設定單局目標分數(11 / 15 / 21 分)、是否需贏兩分以上、單局或三戰兩勝 / 五戰三勝
- **可安裝成手機 App(PWA)**:透過瀏覽器「加入主畫面」後,全螢幕顯示、無網址列,並支援離線使用

---

## 安裝成手機 App

**iPhone(Safari)**
1. 用 Safari 打開網站
2. 點下方分享圖示 → 「加入主畫面」

**Android(Chrome)**
1. 用 Chrome 打開網站
2. 右上角選單 → 「安裝應用程式」或「加入主畫面」

## 技術架構

純前端 HTML / CSS / JavaScript,無框架、無後端依賴:

```
pickleball-scorer/
├── index.html              主程式(介面 + 計分邏輯)
├── manifest.json           PWA 設定(App 名稱、圖示、啟動模式)
├── sw.js                   Service Worker,提供離線快取
├── icon-192.png            App 圖示
├── icon-512.png
└── icon-maskable-512.png
```

- 字型:Oswald(記分板數字)、Inter(介面文字)、IBM Plex Mono(報分格式與細節資訊)
- 比分狀態僅存在單次瀏覽器 session 中(記憶體),重新整理頁面會重置,目前無資料庫串接

## 部署方式

專案使用 [Vercel](https://vercel.com) 部署:

```bash
cd pickleball-scorer
vercel --prod
```
